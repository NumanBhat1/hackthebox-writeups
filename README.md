# HackTheBox Writeups

My notes and walkthroughs from HackTheBox machines I've worked through, written
up the way I'd explain them to a teammate: what I found, how I approached it, and
what I learned. I'm building these as I study for CompTIA Security+ and work
toward a SOC / blue team role.

> **Scope:** These writeups only cover **Starting Point** and **retired** machines,
> in line with HackTheBox's policy. I do not publish writeups for active machines.

## How I write these up

Each writeup follows the same structure so they're easy to read:

1. **Overview** - the machine, tier, and what skills it exercises
2. **Enumeration** - how I scanned and what I found
3. **Gaining access** - the path I took to get in
4. **Findings** - what I accessed (flag values omitted)
5. **What I learned** - the takeaway

Each box has its own folder with its writeup and screenshots.
